git操作步骤：

1、下载安装git
网址：https://git-scm.com/downloads
2、新建一个空白文件夹，在空白文件夹中打开git Bash
3、初始化文件夹
命令：git init 
4、克隆存储库，打开网址https://github.com/prime-rzq/englishClub，复制存储库code（例如：https://github.com/prime-rzq/englishClub.git），然后使用命令：
git clone https://github.com/prime-rzq/englishClub.git
5、拉取数据，使用命令：
git pull
6、此时，社团的数据就被下载下来啦，添加修改数据之后，就可以推送数据啦
git push
至此就完成了资料的更新~
